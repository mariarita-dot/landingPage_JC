# JC Raspagem de Tacos e Assoalhos — Landing Page

Landing page profissional desenvolvida para a **JC Raspagem de Tacos e Assoalhos**, empresa com mais de 30 anos de experiência em restauração de pisos de madeira, atuando em Santos e região, SP.

---

## Sobre o Projeto

Site institucional criado para apresentar os serviços da empresa, gerar contato via WhatsApp e transmitir credibilidade ao cliente. O design foi pensado para valorizar a identidade da marca e facilitar o contato pelo celular.

---

## Funcionalidades

- **Hero section** com imagem de fundo e chamada principal
- **Seção "Por que conosco"** com diferenciais numerados
- **Galeria de serviços** com fotos reais do trabalho
- **Seção Sobre nós** com cards de serviços oferecidos
- **Estatísticas** — anos de experiência, serviços concluídos, satisfação
- **Formulário de contato** com redirecionamento direto para o **WhatsApp**
- **Navegação suave** (scroll) entre seções
- **Animações de entrada** via Intersection Observer
- **Layout totalmente responsivo** para mobile e desktop

---

## Tecnologias Utilizadas

| Tecnologia | Uso |
|---|---|
| React | Framework principal |
| Vite | Bundler e servidor de desenvolvimento |
| CSS puro | Estilização e animações |
| WhatsApp API | Redirecionamento do formulário de contato |

## Como Rodar Localmente

```bash
# Clone o repositório
git clone https://github.com/mariarita-dot/landingPage_JC.git

# Entre na pasta
cd landingPage_JC

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```

Acesse em: `http://localhost:5173`

---

## 📦 Build para Produção

```bash
npm run build
```

Os arquivos prontos para deploy ficam na pasta `dist/`.
